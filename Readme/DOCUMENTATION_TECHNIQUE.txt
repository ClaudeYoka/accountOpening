1️⃣ ARCHITECTURE GÉNÉRALE
═════════════════════════

Le système de sections fonctionne sur le principe MVVM (Model-View-View-Model):

┌─────────────────────────────────────────────────────┐
│                    DOM HTML                         │
│  (form-section-1, form-section-2, etc.)            │
└────────────────┬────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────┐
│              STATE MANAGER                          │
│  currentSection = 1                                 │
│  maxSection = 4                                     │
└────────────────┬────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────┐
│            FUNCTION HANDLERS                        │
│  showSection(num)                                   │
│  nextSection()                                      │
│  prevSection()                                      │
└────────────────┬────────────────────────────────────┘
                 │
                 ▼
┌─────────────────────────────────────────────────────┐
│           VIEW UPDATES                              │
│  • Cache toutes sections                            │
│  • Affiche section active                           │
│  • Met à jour nav                                   │
│  • Scroll automatique                               │
└─────────────────────────────────────────────────────┘

4️⃣ JAVASCRIPT
══════════════

Core IIFE (Immediately Invoked Function Expression):
─────────────────────────────────────────────────────

(function() {
    let currentSection = 1;
    const maxSection = 4;
    
    // Initialization
    window.addEventListener('DOMContentLoaded', function() {
        showSection(1);
    });
    
    // Main Function
    function showSection(sectionNum) {
        if (sectionNum < 1 || sectionNum > maxSection) return;
        
        // Hide all sections
        for (let i = 1; i <= maxSection; i++) {
            const el = document.getElementById('form-section-' + i);
            if (el) el.style.display = 'none';
        }
        
        // Show selected section
        const section = document.getElementById('form-section-' + sectionNum);
        if (section) {
            section.style.display = 'block';
            setTimeout(() => {
                section.scrollIntoView({ behavior: 'smooth', block: 'start' });
            }, 50);
        }
        
        // Update indicators
        document.querySelectorAll('.section-nav-item').forEach(item => {
            item.classList.remove('active');
            if (parseInt(item.dataset.section) === sectionNum) {
                item.classList.add('active');
            }
        });
        
        currentSection = sectionNum;
    }
    
    // Navigation functions
    function nextSection() {
        if (currentSection < maxSection) {
            showSection(currentSection + 1);
        }
    }
    
    function prevSection() {
        if (currentSection > 1) {
            showSection(currentSection - 1);
        }
    }
    
    // Expose to global scope
    window.showSection = showSection;
    window.nextSection = nextSection;
    window.prevSection = prevSection;
    window.currentFormSection = function() { return currentSection; };
    
    // Event listeners
    document.addEventListener('click', function(e) {
        if (e.target.classList.contains('section-nav-item')) {
            const sectionNum = parseInt(e.target.dataset.section);
            showSection(sectionNum);
        }
    });
    
    // Print handlers
    window.addEventListener('beforeprint', function() {
        for (let i = 1; i <= maxSection; i++) {
            const el = document.getElementById('form-section-' + i);
            if (el) el.style.display = 'block';
        }
        const nav = document.getElementById('section-navigation');
        if (nav) nav.style.display = 'none';
    });
    
    window.addEventListener('afterprint', function() {
        showSection(currentSection);
        const nav = document.getElementById('section-navigation');
        if (nav) nav.style.display = 'flex';
    });
})();

6️⃣ API JAVASCRIPT
═════════════════

Public Functions:
─────────────────

window.showSection(num: number) → void
  Afficher une section spécifique (1-4)
  
  Exemple:
  window.showSection(2);  // Afficher Section 2

window.nextSection() → void
  Aller à la section suivante
  
  Exemple:
  window.nextSection();  // Section 1 → Section 2

window.prevSection() → void
  Aller à la section précédente
  
  Exemple:
  window.prevSection();  // Section 2 → Section 1

window.currentFormSection() → number
  Retourner le numéro de section actuellement affichée
  
  Exemple:
  const sec = window.currentFormSection();
  console.log('Section actuelle:', sec);

State Management:
─────────────────
currentSection  (privé) - Section affichée actuellement
maxSection      (privé) - Nombre maximum de sections (4)

Usage in HTML:
──────────────
<button onclick="window.nextSection()">Suivant</button>
<button onclick="window.prevSection()">Précédent</button>
<button onclick="window.showSection(3)">Aller à Section 3</button>


7️⃣ EVENTS
═════════

Print Events:
─────────────

beforeprint
  • Toutes les sections deviennent visibles
  • Navigation disparaît
  • Contenu se réorganise pour impression

afterprint
  • Retour à la vue section unique
  • Navigation réapparaît
  • Section active restaurée

DOM Events:
───────────

click (document-level)
  Écoute les clics sur .section-nav-item
  Appelle showSection() avec le numéro approprié

DOMContentLoaded
  Initialise la première section au chargement
  Lance showSection(1)

Custom Events (possibles):
──────────────────────────

Vous pourriez ajouter:
  const event = new CustomEvent('sectionChanged', {
      detail: { section: newSection }
  });
  document.dispatchEvent(event);

Common Issues:
──────────────

1. showSection is not defined
   → Vérifier que le script s'est exécuté
   → Vérifier les erreurs JS dans la console

2. Section n'apparaît pas
   → Vérifier que l'ID existe: form-section-X
   → Vérifier le CSS: display: none vs block

3. Navigation ne fonctionne pas
   → Vérifier que data-section est correct
   → Vérifier les event listeners

4. Impression ne masque pas navigation
   → Vérifier que @media print est chargée
   → Vérifier les !important flags
